### Cloning a repo:
1.  Get HTTPS github link for repo
2.  Navigate in terminal to where you want your repository to live
3.  In terminal: `git clone <https repo link>`

### Making this repository yours (pointing to your github):
1.  Create a new repo on github
2.  Copy HTTPs string
3.  In terminal of project: `git remote set-url origin <http string>`
4.  Double check remote origin `git remote -v`

### TO delete a repo link on your local machine:
1.  Terminal is in the correct repo
2.  In terminal: `rm -rf .git`
